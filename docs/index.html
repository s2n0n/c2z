c2z Helm Repository

Repository Information

This is the Helm chart repository for c2z (Container to Zone) - A Kubernetes-based penetration testing lab environment.

Add this repository

# add c2z helm repo
helm repo add c2z https://s2n0n.github.io/c2z/
helm repo update

Install a chart

helm install my-c2z c2z/c2z

Available Charts

  • c2z - Main c2z penetration testing lab chart
